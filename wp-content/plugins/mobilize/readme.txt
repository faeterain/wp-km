== Description ==

The **Mobilize** plugin adds a swipe menu to your site, if the width is lower than a certain (configurable) value.

Demo and more details here:
https://getbutterfly.com/wordpress-plugins/ *(hint: use your mobile device)*

== Installation ==

1. Upload to your plugins folder, usually `wp-content/plugins/`
2. Activate the plugin on the plugin screen
3. Configure the plugin from **Settings** -> **Mobilize**
4. Assign a menu to the "Mobilize Navigation" section in **Appearance** -> **Menus**
5. Create Mobilize widgets in **Appearance** -> **Widgets**
